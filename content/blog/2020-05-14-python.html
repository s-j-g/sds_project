R and Python Interaction

Reticulate package allows R and Python to interact with each other.

# R code
library(reticulate)
la<-"indian"
# Python code
import re 
la="blanket"
# R code: access R-defined objects with r
cat(c(la,py$la))
## indian blanket
# Python code: access Python-defined objects with py$
print(r.la,la)
## indian blanket
